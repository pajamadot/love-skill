# 依恋理论框架（鲍尔比 & 安斯沃斯）

## 核心原则
童年早期形成的依恋风格成为影响成人浪漫关系的内部工作模型。理解伴侣的依恋风格可以解锁冲突行为背后的"为什么"。

## 评估指标

### 安全型依恋指标
- 舒适地表达脆弱性和自主性
- 以共情和陪伴回应伴侣的痛苦
- 能承受分歧而不将其解读为对关系的威胁
- 直接表达需求："我需要和你多一些高质量的共处时间"
- 争吵后恢复相对较快
- 假设伴侣是善意的

### 焦虑-沉浸型指标
- 对伴侣的情绪和可用性高度警觉
- 伴侣没有及时回复时反复发消息/打电话
- "我做错了什么？"的冲突导向
- 寻求确认："你还爱我吗？""我们还好吗？"
- 伴侣不在时难以自我安抚
- 核心恐惧：被抛弃

### 回避-疏离型指标
- 强调独立性："我需要我的空间"
- 对情感对话感到不适："你把这件事看得太重了"
- 可能理智化感受：分析而非感受
- 在情感压力下退缩
- 核心恐惧：被吞没/失去自主

### 恐惧-回避（混乱型）指标
- "过来/走开"的摇摆
- 对亲密的强烈渴望与恐惧并存
- 关系太好时可能自我破坏
- 不可预测的情绪反应
- 核心恐惧：被抛弃和被吞没并存

## 焦虑-回避陷阱

最常见的破坏性模式：
1. 焦虑型感知到距离 -> 激活依恋系统 -> 追逐
2. 回避型感知到压力 -> 激活去激活系统 -> 退缩
3. 退缩确认焦虑型的遗弃恐惧 -> 更加追逐
4. 更多追逐确认回避型的吞没恐惧 -> 更加退缩
5. 循环升级直到危机

### 打破陷阱
- 命名循环："我们又在跳追逐-退缩的舞了"
- 焦虑型：在伸出手之前先练习自我安抚
- 回避型：在被要求之前主动提供小的连接举动
- 双方：协商"信号系统"——焦虑型说"我需要确认"，回避型说"我需要20分钟"

## 赢得安全感

依恋风格不是固定的。通过持续的积极关系体验，不安全依恋可以转向"赢得的安全感"：
1. 觉察：识别你的依恋风格及其起源
2. 观察：实时注意依恋行为（不判断）
3. 选择：在触发和习惯反应之间暂停
4. 实验：尝试新的反应
5. 整合：新体验逐渐更新内部工作模型
