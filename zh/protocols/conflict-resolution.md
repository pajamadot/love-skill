# 冲突解决协议（整合版）

## 前提条件
- 双方愿意参与
- 没有安全隐患（虐待、威胁、物质影响）
- 基本共识："我们想解决这个问题，不是为了赢"

## 阶段1：暂停 - 生理调节（5-20分钟）

### 淹没检查
生理淹没（弥漫性生理唤醒/DPA）的信号：
- 心率超过100 BPM
- 浅呼吸或屏住呼吸
- 肌肉紧张（下颌、拳头、肩膀）
- 隧道视野，无法听到伴侣的话
- "我受不了了"的感觉

### 如果被淹没
1. 呼叫结构化暂停："我正在被淹没。我需要[20分钟/一小时]冷静下来。我会在[具体时间]回来继续这个对话。"
2. 暂停期间：散步、呼吸、听音乐。不要排练论点。
3. 自我安抚技术：
   - 4-7-8呼吸法（吸气4秒，屏住7秒，呼气8秒）
   - 渐进式肌肉放松
   - 扎根：说出你看到的5样东西、听到的4样、触碰到的3样
4. 在约定时间返回。

### 反面模式
永远不要在没有返回时间的情况下离开。没有回来承诺的"我要走了"会激活遗弃恐惧并升级冲突。

## 阶段2：倾听 - 结构化对话（15-30分钟）

### 说话者规则
- 使用NVC格式：观察 -> 感受 -> 需求 -> 请求
- 只为自己说话（"我感到..."不是"你让我感到..."）
- 一次只处理一个问题
- 说2-3句后停下来让倾听者回应

### 倾听者规则
- 全部注意力（手机放下，面对伴侣，眼神接触）
- 不打断，不准备反驳
- 复述："我听到你说的是..."
- 问："我理解对了吗？还有更多吗？"
- 确认（确认 ≠ 同意）："你这样感受是有道理的"

### 关键原则
这个阶段的目标是理解，不是同意。你不必同意伴侣才能确认他们的体验。

## 阶段3：分析 - 模式识别（10-15分钟）

### 戈特曼检查
- 四骑士中哪些出现了？
- 可以使用什么解药？
- 修复尝试是否被做出？被接受还是被拒绝？

### 依恋视角
- 哪个依恋系统被激活了？
- "当[事件]发生时，我解读为[含义]，这激活了我对[遗弃/吞没]的恐惧"

### CBT视角
- 是否有认知歪曲在起作用？
- "我告诉自己的故事是...支持和反对这个故事的证据是..."

### 精神分析视角（可选）
- "这让我想起了...我过去的经历"
- "我可能在把我的[品质]投射到你身上"

## 阶段4：搭桥 - 解决方案构建（10-15分钟）

### 绘制循环动态
"我们陷入了一个循环，我[行为]，导致你[行为]，又导致我[行为]..."

### 可行承诺
每位伴侣确定一个小的、具体的、行为性的改变：
- 不是："我会更加支持"（模糊）
- 是："当你下班后压力很大时，我会先问'你想让我听还是帮你解决？'再给建议"（具体、行为性）

### 2%规则
找到你的伴侣立场中你能真诚同意的2%。明确承认它。

## 阶段5：修复 - 重新连接（5-10分钟）

### 修复尝试菜单
至少选一个：
- [ ] 真诚道歉："我为[具体行为]道歉。我知道它[具体影响]。"
- [ ] 承认："我看到我的[行为]伤害了你，那不是我的本意。"
- [ ] 身体重连：牵手、拥抱（如果双方都欢迎）
- [ ] 感谢："谢谢你和我一起留在这个对话中。"
- [ ] 未来承诺："下次这个问题出现时，我想尝试[新方法]。"

### 安排后续
设定具体的检查时间："我们在[日期]重新讨论这个，看看我们双方感觉如何。"

## 紧急降级

当冲突正在升级，无法启动上述协议时：
1. 停止谈论争论的内容
2. 说出正在发生的事："我们在升级。现在我们都听不到对方。"
3. 改变身体状态：站起来、喝水、换个房间
4. 使用最简单的修复："我爱你，我不想伤害你。"
5. 同意在具体的冷却时间后重新讨论
