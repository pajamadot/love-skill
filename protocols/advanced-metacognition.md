# Advanced Metacognitive Architecture for Relationship Analysis

## Purpose
This protocol provides the AI agent with a multi-layered reasoning architecture for analyzing couple dynamics. Each layer adds depth and precision. Apply layers progressively based on complexity.

## Layer 1: First-Order Analysis (Surface)

**What happened?**
Map the observable facts without interpretation:
- Who said what (exact words if available)
- What actions were taken
- What was the context (time, place, stressors)
- What was the sequence of events

**Tool**: Timeline Reconstruction
```
T0: [triggering event]
T1: [Partner A's first response]
T2: [Partner B's reaction]
T3: [escalation or de-escalation point]
T4: [outcome / current state]
```

## Layer 2: Second-Order Analysis (Pattern)

**How does this typically unfold?**
Identify the recurring dynamic:
- Is this a known pattern? (pursue-withdraw, attack-attack, etc.)
- What is the typical trigger?
- What is the typical sequence?
- How does it usually end?
- What has been tried before?

**Tool**: Pattern Frequency Map
```
Pattern: [name the cycle]
Frequency: [daily / weekly / monthly / triggered by specific events]
Duration: [how long does the cycle last once activated]
Recovery: [how long until relationship returns to baseline]
Trend: [escalating / stable / de-escalating over time]
```

## Layer 3: Third-Order Analysis (Structure)

**What beliefs and rules drive the pattern?**
Excavate the operating assumptions:

**For each partner:**
- Relationship rules: "In a relationship, you should..."
- Emotional rules: "It's not okay to feel/show..."
- Gender rules: "As a man/woman/partner, I should..."
- Conflict rules: "When there's conflict, the right thing to do is..."
- Love rules: "If someone really loves you, they would..."

**Tool**: Belief Audit
```
Partner A's Rule: "[belief statement]"
  Origin: [where did they learn this?]
  Function: [what does this rule protect them from?]
  Cost: [what does this rule prevent?]
  Test: [is this rule true in all cases?]
```

## Layer 4: Fourth-Order Analysis (Mental Models)

**What worldview makes these structures inevitable?**
Identify the deepest operating paradigm:

| Mental Model | Implication | Counter-Model |
|---|---|---|
| "Love should be effortless" | Conflict = failure | "Love is a practice requiring ongoing effort" |
| "My partner should complete me" | Unmet needs = partner's fault | "I am whole; my partner enhances my life" |
| "Vulnerability is weakness" | Emotional openness = danger | "Vulnerability is courage and connection" |
| "If I compromise, I lose" | Partnership = zero-sum | "Collaboration creates outcomes neither could alone" |
| "Past predicts future" | Change is impossible | "People can grow; patterns can shift" |
| "I know what's best" | My perspective = reality | "My perspective is one lens among many" |

**Tool**: Mental Model Comparison
```
Partner A's Model: "[worldview]"
Partner B's Model: "[worldview]"
Collision Point: [where these models contradict]
Integration: [what larger model holds both?]
```

## Layer 5: Multi-Perspective Synthesis

**The Seven-Lens Protocol**

For any issue, systematically apply each lens and synthesize:

### Lens 1: Attachment
"What attachment need is activated? What is each partner's nervous system telling them?"

### Lens 2: Developmental
"What developmental stage is this couple in? What growth is trying to happen?"
- Merger (early relationship) -> Differentiation -> Reconnection (mature bond)
- Is conflict arising because one partner is differentiating while the other resists?

### Lens 3: Systemic
"What circular patterns maintain this problem? Where are the leverage points?"

### Lens 4: Intergenerational
"What patterns from family of origin are being replayed? What was modeled?"

### Lens 5: Sociocultural
"What cultural, religious, economic, or social pressures shape this dynamic?"

### Lens 6: Somatic
"What is happening in each partner's body? What does the nervous system need?"

### Lens 7: Existential
"What deeper questions about meaning, mortality, freedom, or identity does this conflict touch?"

### Synthesis
"Which lens reveals the most actionable insight? Which combination creates the deepest understanding?"

## Layer 6: Recursive Self-Reflection

**The agent reflects on its own analytical process.**

After completing the analysis, ask:
1. "Am I unconsciously favoring one partner's perspective? Why?"
2. "Am I applying frameworks that confirm my initial hypothesis, or am I genuinely exploring?"
3. "What am I NOT seeing? What framework might I be missing?"
4. "Am I oversimplifying to fit a model, or honoring the couple's actual complexity?"
5. "Is my recommendation the easiest intervention or the most effective one?"
6. "Would a therapist of a different orientation see this differently? How?"

## Advanced Reasoning Strategies

### Dialectical Triangulation
For every conclusion, generate:
1. The thesis (your primary analysis)
2. The antithesis (the strongest counter-argument)
3. The synthesis (the integration that transcends both)

### Analogical Reasoning
"This couple's dynamic is analogous to [other system]. What insights transfer?"
- International diplomacy (escalation/de-escalation, trust-building, negotiation)
- Ecology (symbiosis, parasitism, mutualism, balance)
- Music (harmony, dissonance, resolution, rhythm, counterpoint)
- Dance (leading, following, attunement, flow, stumbling)

### Counterfactual Analysis
"If [one variable] were different, how would this change?"
- If Partner A had secure attachment, how would they respond?
- If Partner B expressed vulnerability instead of anger, what would happen?
- If this couple had no financial stress, would this conflict exist?
- If they communicated in each other's "language," what would shift?

### Temporal Analysis
Map the issue across three timescales:
- **Micro** (this moment): What just happened in the last interaction?
- **Meso** (this phase): What pattern has been operating over weeks/months?
- **Macro** (this relationship): What is the arc of this relationship's development?

### Embodied Simulation
For each partner, the agent mentally simulates:
"If I were this person, with their history, their attachment style, their cultural background, their current stress level, their body sensations... what would I FEEL right now? What would I NEED? What would I be AFRAID of?"

This is not projection - it's disciplined imaginative empathy constrained by theoretical frameworks.

## Meta-Framework Selection Matrix

When multiple frameworks could apply, use this decision matrix:

| Criteria | Weight | Framework A | Framework B | Framework C |
|---|---|---|---|---|
| Fits the couple's presenting issue | 30% | | | |
| Addresses the deepest layer of the problem | 25% | | | |
| Produces actionable interventions | 20% | | | |
| Matches the couple's readiness for depth | 15% | | | |
| Builds on what's already been tried | 10% | | | |

Score each framework 1-5 on each criterion. Use the highest-scoring framework as primary, second-highest as secondary.

## Error Correction Protocols

### Confirmation Bias Check
"Am I seeking evidence for my initial impression, or actively looking for disconfirming evidence?"

### Anchoring Check
"Am I overly influenced by the first piece of information I received? What if I started from the other partner's account?"

### Availability Bias Check
"Am I applying the framework I know best rather than the one that fits best?"

### Empathy Fatigue Check
"Have I lost curiosity about one or both partners? Am I recommending something generic rather than tailored?"

### Complexity Tolerance Check
"Am I rushing to a clean resolution when the situation requires sitting with ambiguity?"

## Output Quality Criteria

A high-quality relationship analysis should:
1. Validate both partners' experiences (neither feels pathologized)
2. Identify at least one pattern neither partner has named
3. Connect the present conflict to deeper attachment/developmental themes
4. Provide specific, actionable, behavioral recommendations (not just insight)
5. Include a metacognitive tool the couple can use independently
6. Acknowledge what the AI agent cannot see or know
7. Recommend professional help when appropriate
